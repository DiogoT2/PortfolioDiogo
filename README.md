# Diogo Trincheiras - Portfolio Workspace

A beautifully crafted, modern React portfolio site built with Vite, Tailwind CSS v4, and Framer Motion. This project is a major enhancement of the original portfolio, incorporating 30 professional improvements covering Design, Features, Code Quality, and SEO.

## 🚀 Features

### Design & Visual Polish
- Custom fluid cursor with selection snapping
- System-respecting dark/light Theme Toggle
- Dynamic floating "blobs" and Hero background
- Interactive 3D tilt project cards
- Smart Morphing Navbar (pill-shaped on scroll)
- Interactive Timeline for Work & Education
- Infinite scrolling Tech Skills Marquee
- Magnetic Buttons for micro-interactions
- Developer-centric ASCII Loader screen
- Scrambling Text animations

### Features & Functionality
- Full form validation for the Contact Message
- 'Hire Me' availability status pulse
- Intelligent dynamic project category filtering
- Toast notifications utilizing Sonner
- Clean reading progress bar indicator
- Back to Top float button
- Copy-to-clipboard functionality for Email
- Persisted Theme selection in LocalStorage

### Code Quality, Performance & SEO
- Complete Meta and Open Graph tags injected
- Image loading optimizations using `loading="lazy"`
- Code Splitting / Lazy Loading setup via `React.lazy`
- Fully semantic HTML and Screen-Reader friendly components
- Encapsulated custom Error Boundaries
- Clean componentization and strong Type Safety

## 🛠️ Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion**
- **Lucide Icons**
- **Sonner**

## 📦 Getting Started

1. Clone repository
2. Install dependencies: `npm install`
3. Run dev environment: `npm run dev`
4. Build for deployment: `npm run build`

> Built securely and dynamically. No backend database required. Hosted via static providers (Vercel/Netlify/GitHub Pages).
